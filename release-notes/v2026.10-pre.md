OPM Flow for Windows — native MSVC build (hybrid MPI + OpenMP) with a Qt GUI.

**Downloads** — the installer and the portable zip both work for most people; pick whichever you prefer.
- `OPM-Flow-2026.10-pre-Setup.exe` — installer: adds Start-menu shortcuts, and if the VC++ or MS-MPI runtimes are missing it installs them for you.
- `opm-flow-2026.10-pre-win64.zip` — portable: unzip and run `bin\flow-gui-qt.exe`. If the VC++ / MS-MPI runtimes aren't already installed, run the two installers in `redist\` once first.
- `OPM-Flow-2026.10.0.0.msix` — unsigned MSIX (for sideloading/Store experiments; see PACKAGING.md). **Note:** Windows will not enable its **Install** button unless the signing certificate is already trusted on the machine (it shows *"Publisher: Unknown / this app package's certificate could not be verified"*).

**Contents:** `flow.exe` (all model variants, including black-oil), `flow-gui-qt.exe` (job queue GUI), all required runtime DLLs.

**Getting past SmartScreen (this build is unsigned)**
The binaries are not code-signed, so Windows shows *reputation* warnings — these are not virus detections, and for the `.exe`/`.zip` they are dismissible:
- **When downloading** (Edge/Chrome): if it says *"… isn't commonly downloaded"* and the download stops or shows **Canceled**, open the download's **⋯ (More actions)** menu → **Keep**, then **Keep anyway**.
- **On first launch**: if *"Windows protected your PC"* appears, click **More info → Run anyway**.

**Notes**
- Windows 10/11 x64.
- Parallel runs: `mpiexec -n <ranks> flow.exe DECK.DATA --threads-per-process=<threads> --output-dir=<dir>`, or use the GUI.
